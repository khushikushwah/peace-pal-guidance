# 🌿 Welcome to Aethel — A Stress-Free AI Companion

*A Stress-Free AI Companion to help you reflect, relax, and restore calm.*

---

## ✨ Overview
Serene Companion is a web app designed to gently guide users toward peace of mind.  
Through mood-based conversations, calming recipes, and curated music, Aethel creates a sanctuary where you can share what weighs on you and find relief.

## ✨ Features
- 🧘 **Mood Selector** — Choose from moods like *Seeking Stillness* or *Drifting* to personalize your experience.  
- 🎤 **Voice Orb** — Speak freely and let Aethel listen with gentle guidance.  
- 📚 **Suggestion Grid** — Explore calming prompts and reflections.  
- 🍲 **Recipes Section** — Six stress‑relief recipes with expandable ingredients & methods.  
- 🎶 **Songs Section** — Eight curated mind‑relief tracks with inline playback.  
- 🔔 **Notifications** — Get notified when tasks complete or input is needed.  
- 🛡️ **Security Enhancements** — Input validation, mood allow‑list, and safe error handling.  

---

## 🛠️ Tech Stack
- **Frontend:** React + Vite + TypeScript  
- **Styling:** Tailwind CSS  
- **Backend:** Supabase Functions (`aethel-chat`)  
- **Integrations:** YouTube inline playback, Supabase database  

---
